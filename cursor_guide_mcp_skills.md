# Шпаргалка: MCP и Skills

## 1. MCP-серверы (Инструменты ИИ)
**Статус:** Доступны всегда, но **нужно просить словами** их использовать.

* **Context7** (Актуальная документация): *"Используй Context7 для поиска свежих доков по ROS 2."*
* **Sequential Thinking** (Логика): *"Используй sequential-thinking, чтобы пошагово найти баг."*
* **Playwright** (Браузер): *"Используй Playwright, зайди на localhost:8080 и нажми кнопку."*
* **WCGW** (Терминал-агент): *"Запусти wcgw для автономной сборки с исправлением ошибок."*

## 2. Skills (Навыки / папка `.cursor/skills/`)
**Статус:** Срабатывают **АВТОМАТИЧЕСКИ**.

* **Как работают:** Вы просто ставите задачу. Если запрос совпадает с описанием навыка, ИИ сам применяет его.
* **Пример:** Вы пишете *"Собери проект"*. ИИ молча читает навык `ros2-project-setup` и делает `colcon build` как вы его учили.
* **Как создать:** Попросите ИИ: *"Создай навык для оформления пулл-реквестов."*
